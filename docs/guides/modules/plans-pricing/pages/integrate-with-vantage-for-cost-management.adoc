= Integrate with Vantage for cost management
:page-platform: Cloud
:page-description: Connect CircleCI to Vantage to track CI/CD costs alongside your cloud infrastructure and SaaS providers.
:experimental:

Vantage provides cost tracking and usage reporting for CircleCI, allowing you to monitor CI/CD spending alongside your cloud infrastructure and SaaS tools in a centralized platform. The integration automatically imports your CircleCI usage data and converts credits into USD-based cost reports.

== Introduction

The Vantage integration enables you to:

* Track CircleCI credit consumption and costs in a unified cost management platform.
* Filter and group costs by organization, project, resource class, and job name.
* Allocate CI/CD expenses to teams using Virtual Tags.
* Identify usage anomalies and spending trends.
* Correlate CircleCI cost changes with related infrastructure shifts.

Vantage automatically imports up to six months of historical data when you first connect, with ongoing imports following your configured retention settings.

This guide provides instructions for connecting CircleCI to Vantage and configuring cost tracking and allocation.

=== How the integration works

The following diagram illustrates how Vantage pulls CircleCI usage data through the API for cost analysis:

[mermaid]
----
flowchart TD
    A[CircleCI pipelines consume credits to run jobs] --> D[CircleCI usage data<br/>tracked by usage type:<br/>Compute, Storage,<br/>Network, IP Ranges, DLC]

    E[Vantage Platform] -->|Call| G[CircleCI Usage Export API<br/>createUsageExport]

    D -.->|Available via API| G

    G -->|Return structured data| I[Usage report in credits broken down by:<br/>Organization, Project,<br/>Resource Class, Job Name,<br/>Build Status, VCS info]

    I --> J[Vantage<br/>Cost Reports & Dashboards]

    J -->|Filter and group by| N[Organization<br/>Project<br/>Job Name<br/>Resource Class<br/>Build Status]
    J -->|Allocate with| O[Virtual Tags<br/>Team, Department,<br/>Business Unit]
    N --> P[Cost analysis to generate insights into:<br/>Spending Trends<br/>Usage Anomalies<br/>Budget Tracking]
    O --> P
----

The integration works by having Vantage call the CircleCI Usage Export API to retrieve your usage data in credits. CircleCI tracks all usage by type (compute, storage, network, and other services), and the API provides structured reports broken down by organization, project, resource class, job name, and build status. Vantage converts your credit usage to USD based on the price per credit you configure, then displays the data in cost reports and dashboards where you can filter, group, and analyze spending trends.

[#connect-vantage]
== 1. Generate a CircleCI personal API token

NOTE: CircleCI personal API tokens do not support fine-grained permission scoping. The token you create will have full access to your organization's data.

To connect Vantage to CircleCI, you need to create a personal API token.

. Log in to the link:https://app.circleci.com[CircleCI web app].
. Select your avatar at the top right of the screen and select **User Settings** from the menu.
. Select **Personal API Tokens**.
. Select btn:[Create new token].
. Provide a descriptive name for the token (for example, "Vantage Integration").
. Select **Add API Token**.
. Copy the token right away, as you cannot view it again.

[#configure-integration]
== 2. Configure the integration in Vantage

Connect your CircleCI organization to Vantage using your personal API token and organization ID.

. Log in to your Vantage account.
. Navigate to **Settings**, then select **Integrations**.
. Locate and select **CircleCI** from the available *Data Sources*.
. Select *Add Organization*.
. Complete the integration form with the following information:
+
* **CircleCI API token**: Paste the token you created in step 1.
* **Organization ID**: Enter your organization ID.
* **Description**: Provide a name to identify this connection (optional).
* **Price per credit in USD**: Enter your organization's credit rate to convert CircleCI credits into US dollars. The standard rate is $.0006 per credit.

. Select **Connect Account**.
. Monitor the import status on the integrations page.

Vantage begins importing your CircleCI usage data. The initial import includes up to six months of historical data, depending on your Vantage retention settings.

[#reporting-capabilities]
== 4. View cost reports

After the integration completes, you can create cost reports to analyze your CircleCI spending.

Cost reports in Vantage support filtering and grouping by:

* **Billing Account**: Your CircleCI organization name.
* **Account**: CircleCI project name.
* **Service type**: Compute, Storage, Users, Network, DLC, Lease, or IP Ranges.
* **Category and Subcategory**: Resource class information.
* **Resource**: Job name.

Vantage automatically creates provider metadata tags for more granular filtering, including:

* Organization ID.
* Project ID.
* VCS details (branch, repository name, repository URL).
* Pipeline ID.
* Build status.

You can use these dimensions to allocate costs to specific teams, track spending by project, or identify optimization opportunities.

Refer to the link:https://docs.vantage.sh/cost_reports[Vantage Cost Reports documentation] for detailed configuration instructions.

[#use-virtual-tags]
== 5. Allocate costs with Virtual Tags

Use Vantage Virtual Tags to allocate CircleCI expenses to teams and detect usage anomalies.

Virtual Tags allow you to:

* Create custom cost allocation rules based on CircleCI metadata.
* Group costs by team, department, or business unit.
* Track spending across multiple projects and organizations.
* Set up alerts for unusual spending patterns.

Refer to the link:https://docs.vantage.sh/virtual_tags[Vantage Virtual Tags documentation] for detailed configuration instructions.
